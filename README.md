# GTBB Discord Bot

Instructions:
- Install: `npm install`
- Register commands: `node src/utils/deploy-commands.js`
- Start bot: `node src/index.js`
- Edit `.env` with your secrets.

For development:
- Clear registered commands: `node src/clear-commands.js`
- Clear database: `node src/clear-db.js`
